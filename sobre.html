Sobre o Projeto

Este site foi criado para o projeto final de Introdução a Programação Web.

Este projeto consiste em servir de Biblioteca e Fórum sobre jogos, onde as pessoas podem ver os jogos que mais aguardam jogar
tal como os que já jogaram. E podem adicionar muitos mais!

Os utilizadores podem filtrar por generos o que é muito facilitador uma vez que as bibliotecas vem a se tornar enormes.
Também podem filtrar por plataforma!

Tecnologias utilizadas

HTML
CSS
JavaScript
PHP

O porquê desta escolha de projeto?

Este site têm como objetivo ser uma biblioteca de jogos.
É literalmente um site onde as pessoas podem procurar pelos seus jogos preferidos e partilhar com as outras pessoas.
