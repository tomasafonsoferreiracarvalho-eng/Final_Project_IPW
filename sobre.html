Sobre o Projeto

Este site foi criado para o projeto final de Introdução à Programação Web.

Consiste, basicamente, numa Biblioteca de jogos, permitindo às pessoas visualizarem alguns jogos novos ou jogos já conhecidos.

Esta Biblioteca permite também, acrescentar/adicionar jogos, do agrado de cada um.

Os utilizadores podem filtrar por géneros o que é muito facilitador, uma vez que as Bibliotecas abrangem uma grande quantidade de informação correspondendo às necessidades de cada um neste caso de jogos.
Um aspeto a salientar é que podem filtrar por plataforma!
